📊 Trading Journal (Web App)

A lightweight trading journal web app to track Forex, Gold, Oil, and Crypto trades with real-time stats, equity curve, and export options. Built with HTML, CSS, and Vanilla JS — no server required, everything runs in your browser.

🚀 Features

Add, edit, and remove trades easily

Auto P/L calculation (Forex, Gold, Oil, Crypto)

Trade categories: All / Forex / Gold / Oil / Crypto

Dashboard with:

✅ Net P/L

✅ Winrate

✅ Avg R:R

✅ Trade count

Charts (Equity curve, Win/Loss ratio, Monthly P/L) using Chart.js

Local storage persistence (your data stays in your browser)

Export trades to PDF

Upload trade screenshots (optional)

🔒 Privacy Options

All trades are stored locally in your browser using localStorage
